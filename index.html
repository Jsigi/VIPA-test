SQL-Definition

Produktive Tabellenstruktur

Die vollständige SQLite-Struktur findest du in sqlite_schema.sql.

-- Auszug aus sqlite_schema.sql
-- Fremdschlüssel aktivieren
PRAGMA foreign_keys = ON;

CREATE TABLE owners (
  id INTEGER PRIMARY KEY,
  name TEXT NOT NULL,
  email TEXT NOT NULL UNIQUE
);

CREATE TABLE dogs (
  id INTEGER PRIMARY KEY,
  owner_id INTEGER NOT NULL REFERENCES owners(id) ON DELETE CASCADE,
  name TEXT NOT NULL,
  breed TEXT,
  vaccinated BOOLEAN DEFAULT false,
  next_vaccine_due DATE
);

Technologien

Stack-Empfehlung

Alles, was du für die Umsetzung benötigst.

  • Frontend: HTML, CSS, JavaScript (hier Vanilla; React/Vue optional)
  • Backend: Node.js oder Python (Express/FastAPI) für REST- oder GraphQL-APIs
  • Datenbank: PostgreSQL oder SQLite für relationale Daten & Foreign Keys
  • Storage: S3/Blob Storage oder Datenbank-Files für Dokument-Uploads
  • Auth: JWT-basierte Session oder OAuth2, falls Zugänge nötig sind
  • Infra: Docker-Container für lokale Entwicklung & Deployment
